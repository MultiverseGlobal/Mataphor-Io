# Metaphor IO: The Universal Cognitive Infrastructure

## The Core Essence
**A universal protocol for idea transmission** — the infrastructure layer for how meaning moves from one mind to another.

## The Insight
Communication doesn't fail because people lack words. It fails because the **cognitive packaging** of an idea — the internal structure, the metaphors, the associative map the sender holds — never fully survives transfer. The receiver reconstructs something adjacent, not the original.

## The Mission
To build a **cognitive commons** — a shared representational substrate where ideas are encoded in a form that preserves their internal architecture across transfer.

## The Infrastructure Layer
Metaphor IO is not an app sitting on top of communication. It is the layer *beneath* all communication. 
**IO** (Input/Output): The interface between a mind and the world.

> "You're not transmitting words. You're transmitting the shape of a thought."

---

### Roadmap: From MVP to 2028
1. **The Bridge (V1)**: Seamless context transfer between existing Large Language Models.
2. **The Schema (V2)**: Formalization of the "Cognitive Packaging" protocol.
3. **The Commons (V3)**: A distributed representational substrate for human-AI-human thought transmission.
4. **The Infrastructure (2028+)**: Universal integration into the interface of thought.
